NCEP/EMC 4KM Gridded Data (GRIB) Stage IV Data
Dataset: UCAR/EOL archive 21.093
DOI: https://doi.org/10.5065/D6PG1QDD
Source page: https://data.eol.ucar.edu/dataset/21.093

Human summary
- Stage IV is a quality-controlled precipitation analysis built from RFC Stage III mosaics.
- Products are available as hourly, 6-hourly, and 24-hourly accumulations.
- Spatial coverage includes CONUS and, for later years, Alaska and Puerto Rico.
- Around July 2020, standard distribution moved from GRIB1 naming to GRIB2 naming.

Why this file is here
- The Erin workflows in this repository use Stage IV as the observed-rainfall reference.
- This text is a shortened provenance note so users know exactly which archive and DOI the data came from.
